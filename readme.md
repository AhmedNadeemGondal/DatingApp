# DatingApp

DatingApp is a full-stack social networking application featuring real-time messaging, photo management, and a robust filtering system. It is built using **ASP.NET Core** for a powerful RESTful API and **Angular** for a dynamic, responsive frontend.

## Features

- **Real-time Communication**: Instant messaging and presence tracking using **SignalR**.
- **Secure Authentication**: Identity management with **ASP.NET Core Identity** and **JWT (JSON Web Tokens)**.
- **Photo Management**: Image uploads and cloud storage integration with **Cloudinary**.
- **User Interactions**: Advanced filtering, sorting, "Likes" system, and member profiles.
- **Modern UI**: Responsive design styled with **Tailwind CSS** and **DaisyUI**.
- **Data Persistence**: Relational data management using **Entity Framework Core** and **SQL Server**.

---

## Technologies Used

### Backend (.NET 9)
- **ASP.NET Core Web API**: High-performance RESTful services.
- **Entity Framework Core**: ORM for SQL Server integration.
- **Identity & Security**: JwtBearer authentication and Bcrypt-based password hashing.
- **CloudinaryDotNet**: Third-party API integration for media storage.

### Frontend (Angular 20)
- **Angular SPA**: Modern component-based architecture.
- **SignalR Client**: Real-time socket connections for chat.
- **Tailwind CSS & DaisyUI**: Utility-first styling and pre-built UI components.
- **RxJS**: Reactive programming for asynchronous data streams.
